# 2048 Clone for Max

This is a library that implements 2048. It's playable from the console, but the real point is to use as a library for an game playing agent. 

Usage

1. Create a board

b = Board()

2.  Eaxh move is just a call to the four move functions. 

1. b.left()
2. b.right()
3. b.up()
4. b.down()
